Privacy Policy

1. Information We Collect

a) Message Data

  • Up to 140 messages per channel for conversation context
  • Message content, timestamps, and associated metadata

b) User Information

  • Discord user ID
  • Display name
  • Server-specific settings
  • Usage statistics and message counts
  • Achievement and badge data
  • User behavior analysis data

2. How We Use Your Information

a) Core Functionality

  • Providing AI-powered responses and assistance
  • Maintaining conversation context
  • Personalizing user experience
  • Processing image analysis and generation requests
  • Analyzing user behavior to improve interactions

b) Service Improvement

  • Training and improving AI models
  • Enhancing bot features and capabilities
  • Refining user behavior analysis algorithms

3. User Rights

You have the right to:

  • Export your chat history
  • Delete your conversation history
  • Modify your user preferences
  • Access your stored data
  • Request information about your behavior analysis data

4. Changes to Privacy Policy

We reserve the right to update this privacy policy. Users will be notified of significant changes through Discord or bot announcements.
